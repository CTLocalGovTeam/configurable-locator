How to use the Event Locator

The Event Locator application can be used by citizens and visitors to discover activities and events occurring in a community.

To locate an event, simply enter an address, or search using a range of dates in which you are interested. Or, you can use your current location to find an event or activity near you. The event or activity will then be highlighted on the map and relevant information presented to you.

Using the Application

The Event Locator application provides residents and visitors an easy way to find events or activities near them, or events taking place on specific dates. Enter your address or click on the Current Location button to find events near you. If you click the Current Location button, it will use the location of your computer's IP address or your cell phone/tablet location to locate events near you. When you enter an address or use your current location, the application will present a list of events and activities near you and allow you to review relevant information about each.

Once you have found an event or activity you are interested in, click the "Add to My List" button in the info-popup window or in the information panel. Once events have been added to your list, you can order them by the dates on which they occur, calculate a route between them, add events to your personal calendar or print your list of events for later use.

Finding an Event of Interest

There are three primary ways to identify an event of interest. You can interact directly with the map and click on an event when you've located it. You can also use your current location or an address to find an event or activity near you. Finally, you can search for an event by using the date picker to select a range of dates you are interested in.

Map Based Navigation

There are several ways to interact with the map:

1. Use the basemap switcher to select the basemap you'd like to interact with
2. Hold down the shift key and drag a box on the map to zoom in
3. Hold Down Ctrl+Shift to zoom out
4. Roll your mouse roller to zoom in/out
5. Click the + or - to zoom in/out

Searching

The search box can be used to find an event or activity you're interested in:

1. Enter the address or location you are interested in
2. Click on the item in the list to locate the address

You can enter a partial address and then identify the correct value in the list.

The Events panel can be used to search for events by date:

1. Use the date picker to define the range of dates you are interested in
2. Click on event in the results list to view related information

Events and Activities

When you find an event you're interested in, you can select it from the list and it will then be highlighted on the map. Additional detail about each event and activity will be provided in the information panel. When you click on an individual event on the map, the same information will be displayed. Click the "Add to My List" button to add the event to a list of events you are interested in. You can then view that list by clicking the "My List" button. Once events have been added to your list, you can order them by the dates on which they occur, calculate a route between them, add events to your personal calendar or print your list of events for later use.

Directions

You can also get directions to each event using your current location or an address you select. When you find an event you'd like to attend, enter an address to start from, or click the use current location button, to calculate the directions and route.

Share Your Map

You can click on the Share Map button to share your map several different ways. You can compose an email, post to your Facebook wall or post a Tweet. When you use the Facebook or Twitter links, you'll be asked to sign in to your respective account.

More Information

The Event Locator application can used by government organizations responsible for showcasing and promoting events and activities to deploy a web-based event locator application. This application provides 24x7 access to the organization and typically supplements customer service phone numbers staffed by these organizations.
